# Color Space Interactive Demo

Interactive color-space learning materials and demos for CIE XYZ, xyY, CIELAB, ICtCp, RGB/CMY mixing, transfer characteristics, HDR/SDR concepts, ICC profiles, and related seminar notes.

## Demo

GitHub Pages entry point:

- `index.html` — interactive CIE color-space demo

## Local Preview

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080/
```

The demo uses CDN-hosted Three.js files and local scripts under `cie_demo/`.
